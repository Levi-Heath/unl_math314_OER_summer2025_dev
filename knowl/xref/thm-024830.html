Theorem 8.8.3.

A symmetric matrix \(A\) is positive definite if and only if \(\mathbf{x}^{T} A \mathbf{x} \gt 0\) for every column \(\mathbf{x} \neq \mathbf{0}\) in \(\R^n\text{.}\)

Proof.

\(A\) is symmetric. Using the Real Spectral Theorem, let
\begin{equation*} Q^{T}AQ = D = \mbox{diag}(\lambda_{1}, \lambda_{2}, \dots, \lambda_{n}) \end{equation*}
where \(Q^{-1} = Q^{T}\) and the \(\lambda_{i}\) are the eigenvalues of \(A\text{.}\) Given a column \(\mathbf{x}\) in \(\R^n\text{,}\) write
\begin{equation*} \mathbf{y} = Q^{T}\mathbf{x} = \left[ \begin{array}{cccc} y_{1} \amp y_{2} \amp \dots \amp y_{n} \end{array}\right]^T. \end{equation*}
Then
\begin{align} \mathbf{x}^TA\mathbf{x} \amp = \mathbf{x}^T(QDQ^T)\mathbf{x} \notag\\ \amp = \mathbf{y}^TD\mathbf{y} \notag\\ \amp = \lambda_{1}y_{1}^2 + \lambda_{2}y_{2}^2 + \dots + \lambda_{n}y_{n}^2. \tag{8.8.1} \end{align}
If \(A\) is positive definite and \(\mathbf{x} \neq \mathbf{0}\text{,}\) then \(\mathbf{x}^{T}A\mathbf{x} \gt 0\) by (8.8.1) because some \(y_{j} \neq 0\) and every \(\lambda_{i} \gt 0\text{.}\)
Conversely, if \(\mathbf{x}^{T}A\mathbf{x} \gt 0\) whenever \(\mathbf{x} \neq \mathbf{0}\text{,}\) let \(\mathbf{x} = Q\mathbf{e}_{j} \neq \mathbf{0}\) where \(\mathbf{e}_{j}\) is column \(j\) of \(I_{n}\text{.}\) Then \(\mathbf{y} = \mathbf{e}_{j}\text{,}\) so (8.8.1) reads \(\lambda_{j} = \mathbf{x}^{T}A\mathbf{x} \gt 0\text{.}\)
